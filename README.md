# 🚀 7datawin - The Ultimate Data Selling App

Ku soo dhawaow **7datawin**! Waa barnaamij (App) casri ah oo loogu talagalay in lagu iibiyo xirmooyinka internet-ka (Data) qiimo jaban oo ku habboon dadka danyarta ah. 📱💸

Waxaa si gaar ah loogu habeeyay suuqa geeska Afrika (Somaaliya, Somaliland, Ethiopia, Kenya) isagoo taageeraya shirkadaha isgaarsiinta ee ugu waaweyn!

---

## ✨ Features (Waxyaabaha uu leeyahay)

- 🎨 **Premium UI/UX:** Naqshad qurux badan oo u eg Mobile App casri ah (Mobile-first design).
- 🌙 **Dark Mode Support:** Taageerada habka madow si indhaha loogu naxo.
- 🛍️ **Shirkado Badan:** Hormuud, Somnet, Telesom, Amtel, Golis, iwm.
- 💳 **OTP & Xaqiijin:** Nidaam ammaan ah oo lagu hubinayo lacag bixinta (Net+ Style).
- 📢 **Xayeysiisyo:** Qeyb u gaar ah dalacsiinta iyo xayeysiisyada (Ad Banners).

---

## 🛠️ Technologies Used (Farsamada la isticmaalay)

- **Frontend:** React.js ⚛️
- **Styling:** Tailwind CSS 🖌️ + Radix UI
- **Icons:** Lucide React 🖼️
- **Routing:** React Router DOM 🛣️

---

## 🚀 Sida loo daarto (Local Development)

Si aad ugu shaqeyso mashiinkaaga (Local), raac talaabooyinkan:

1. **Gal galka frontend-ka:**
   ```bash
   cd app/frontend
   ```

2. **Soo deji baakadaha (Install dependencies):**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Daaran App-ka:**
   ```bash
   npm start
   ```

App-ku wuxuu ka furmi doonaa `http://localhost:3000` 🌐

---

## 🌍 Vercel Deployment (Sida loogu xiro Vercel)

Si aad App-kan asxaabtaada ula wadaagto, waxaad si bilaash ah ugu xiri kartaa [Vercel](https://vercel.com/):

1. Ku shub code-kan **GitHub**.
2. Gal **Vercel** oo samee **New Project**.
3. Dooro kaydkaaga (Repository) ee GitHub.
4. **MUHIIM:** Qeybta `Root Directory`, ka dooro galka `app/frontend` (maxaa yeelay React-gu halkaas ayuu ku jiraa).
5. Guji **Deploy** 🚀

Hadda wixii ka dambeeya, markasta oo aad GitHub update ku sameyso, Vercel si toos ah ayuu isku cusbooneysiin doonaa! 🎉
